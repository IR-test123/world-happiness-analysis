# 🌍 What Makes Countries Happy?

## Overview
This project analyzes global happiness data to understand what factors influence well-being across countries.

## Dataset
World Happiness Report (Kaggle)

## Key Questions
- Leaderboard - The happiest countries
- Irregularities based on survey factors (Dystopia residual) 
- What factors correlate most with happiness?
- Are richer countries happier?
- How has happiness changed over time?

## Tech Stack
- Python
- Pandas
- Matplotlib / Seaborn

## Project Structure
- data/ → raw & processed data
- notebooks/ → exploration
- src/ → reusable code
